Materiales del curso

HTML5

HTML5 permite crear la estructura de una página web mediante etiquetas como encabezados, párrafos, imágenes, tablas, listas, formularios y secciones.

CSS3

CSS3 se utiliza para dar diseño y presentación al sitio web. Gracias a CSS podemos cambiar colores, tamaños, márgenes, fondos, bordes y adaptar el diseño a distintos dispositivos.

Enlaces relativos y absolutos

Los enlaces relativos permiten conectar páginas internas del mismo proyecto, mientras que los enlaces absolutos dirigen a páginas externas como sitios web de consulta y herramientas de desarrollo.

Formularios y multimedia

En el curso también se trabaja con formularios, imágenes, videos y recursos descargables, elementos muy importantes para desarrollar páginas web completas.
